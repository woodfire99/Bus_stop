# Bus_stop

## 설명(Description)
개발기간(Development Period) : 2022年 5月〜6月  

Android + Google MAPS API + 공공데이터 API

Android Studio를 활용하여 공공데이터포털의 버스 관련 Open API를 연동한 버스 정류장 정보 안내 앱을 개발하였습니다. 사용자는 현재 위치 주변의 버스 정류장 목록을 확인하고, 특정 정류장을 선택하면 실시간 도착 정보 및 경유 노선을 조회할 수 있습니다.

(Android + Google Maps API + Public Data API

Developed a bus stop information app using Android Studio by integrating public transportation Open APIs provided by the Korean government.
Users can view a list of nearby bus stops based on their current location, and upon selecting a specific stop, they can check real-time arrival information and bus routes passing through that stop.)
